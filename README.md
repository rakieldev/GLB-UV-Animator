# GLB UV Animator Tool

Tool for Adding UV Animations to GLB Models Used in Ikemen GO

# Usage:

1.Open a GLB model. Its nodes will be loaded automatically.  
2.Find the node where you want to apply the animation.  
3.Set the animation speed and direction.  
4.Choose an output directory (optional). By default, the file will be saved in the same directory as the original model.  
5.Click Apply Anim.  

# Notes:

- These animations are generally meant to be applied at the final stage of stage creation.
- If you open the GLB file in Blender, you won’t be able to see the animations.
- To apply animations to multiple nodes, open the model that already has an animation applied and run the tool again on another node.
